================================================================================
  ALBORADA BOX — INSTALLATION OVERVIEW
  IPTV Streaming Platform (Laravel + Docker + XUI.ONE) — Two-Server Setup
================================================================================

The platform is deployed on TWO separate servers, each with its own setup
script and its own install guide:

  ┌───────────────────────────────────────────────────────────────────────┐
  │  SERVER 1 — WEB SERVER                                                │
  │    Script : websetup.sh                                               │
  │    Guide  : install-web.txt                                           │
  │    Spec   : Ubuntu 22.04/24.04 · 2 vCPU · 4 GB RAM · 40 GB SSD        │
  │    Runs   : Laravel web panel (Docker) · Nginx + SSL · MySQL          │
  │             phpMyAdmin (https://yourdomain.com/phpmyadmin)            │
  │             FREE self-hosted email server (Postfix + OpenDKIM)        │
  ├───────────────────────────────────────────────────────────────────────┤
  │  SERVER 2 — STREAMING SERVER                                          │
  │    Script : streamsetup.sh                                            │
  │    Guide  : install-stream.txt                                        │
  │    Spec   : Ubuntu 22.04 ONLY · 4–8 vCPU · 8–16 GB RAM · 100 GB+ SSD  │
  │    Runs   : XUI.ONE 1.5.13 IPTV panel · MariaDB                       │
  └───────────────────────────────────────────────────────────────────────┘

RECOMMENDED INSTALL ORDER
  1. Install the STREAMING server first  → install-stream.txt
     (so its IP is known when configuring the web panel)
  2. Install the WEB server              → install-web.txt
  3. Connect them: web Admin Panel → Settings → IPTV
       xtream_base_url           = http://STREAMING_SERVER_IP:8080
       xtream admin credentials  = your XUI.ONE admin login
       iptv_provisioning_enabled = 1
  4. Add the email DNS records printed by websetup.sh
       cat /root/.alborada_dns_records   (on the web server)
  5. Configure Stripe and test a purchase end-to-end (install-web.txt §7–8)

DNS RECORDS AT A GLANCE
  Type   Host              Value                    Server
  ─────────────────────────────────────────────────────────────
  A      @                 <WEB_SERVER_IP>          web
  A      www               <WEB_SERVER_IP>          web
  A      mail              <WEB_SERVER_IP>          web (mail server)
  A      stream            <STREAMING_SERVER_IP>    streaming
  MX     @                 mail.yourdomain.com      web (mail server)
  TXT    @ / mail._domainkey / _dmarc               web — values generated
                                                    by websetup.sh into
                                                    /root/.alborada_dns_records

  Cloudflare: orange cloud (proxy) for @ and www only;
  grey cloud (DNS-only) for mail and stream.

PORT MAP
  Web server       : 80/443 public · 8081 web container (localhost) ·
                     8082 phpMyAdmin (localhost) · 25 Postfix (Docker-only)
  Streaming server : 8080 XUI.ONE admin · 2086 customer streams ·
                     25461 XUI.ONE internal

LEGACY SINGLE-SERVER SETUP
  autosetup.sh (with install instructions in its header comments) installs
  everything on ONE VPS. It is kept for small/test deployments, but the
  two-server setup above is the supported production layout. Note that
  autosetup.sh does NOT install phpMyAdmin or the local email server.

ESTIMATED MONTHLY COSTS
  Web server             : $16–24/mo
  Streaming server       : $40–60/mo
  Domain name            : ~$1/mo ($12/yr)
  Email server           : $0 (self-hosted Postfix on the web server)
  Stripe fees            : 2.9% + $0.30 per transaction
  Content/streams        : $150–400 (one-time or monthly reseller credits)
  ────────────────────────────────────────────
  Infrastructure total   : ~$57–85/mo before content costs
